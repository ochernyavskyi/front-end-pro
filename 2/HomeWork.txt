1.	false && true || true
Будет true.  Если в правой части получается true, то и все значение будет true

2.	false && (true || true);
Будет false. Так как используется оператор И, а так как в левой и правой части операнды не равны, то будет false

3.	+'40' + +'2';
Будет 42. Если два операнда строки имеющие числа, и оператор плюс, то они приведутся в числовой формат и суммируются.

4.	'2' + 3 ** 2;
Будет 29. Сначала произойдет возведение в степень, а потом произойдет конкатенация.

5.	3 ** 2 / 3;
Будет 3. Сначала выполняется возведение в степень, потом происходит деление

6.	3 ** (9 / 3);
Будет 27. Сначала выполняется действие в скобках, потом происходит возведение в степень.

7.	'10' + (5 === 6);
Будет 10false, так как 5 не равное 6 – это false, а знак + конкатенирует значения

8.	'10' - 5 === 6;
Будет false. При любых обстоятельствах приведения строгое равенство достигнуто не будет

9.	undefined + 1;
Будет NaN. Любые действие с undefined приведёт значение выражения к NaN

10.	(+null == false) < 1;
Будет false. Так как null приведётся к 0 и false тоже, соответственно 0 равен 0, а это значит значение true, так как true это 1, и утверждение, что 1<1 является ложью

11.	(+null == false) < 1 ** 5;
Будет false. Так как null приведётся к 0 и false тоже, соответственно 0 равен 0, а это значит значение true, так как true это 1, а 1 в пятой степени тоже 1, то значение будет false.

12.	var number = 5;
2 * ++number;
2 * number--;
console.log(number);
Будет 5, так как мы новые значения не присваивали переменной number
